# NEURA — Brand Guidelines

## Principe fondateur
Le brandmark N joue un double rôle :
- **Lockup horizontal** : `[N-signe]eura` — une seule entité graphique continue
- **Signe isolé** : favicon, app icon, tampon

Le N n'est jamais répété. Le signe EST la lettre.

## Construction du brandmark N
- Grille carrée, ratio hauteur/largeur **4:3**
- Deux jambages verticaux (parallelogrammes, terminaisons 45°)
- Diagonale remplacée par un **arc de cercle** (compas) → lecture N + signal
- Monoline strict : même épaisseur jambages = épaisseur arc
- Reconstructible avec : règle + compas + rapporteur

## Palette chromatique

| Token            | Hex       | Usage                          |
|------------------|-----------|--------------------------------|
| Noir profond     | `#0A0A0A` | Background principal           |
| Cyan électrique  | `#00E5FF` | Accent — signe N uniquement    |
| Blanc pur        | `#FFFFFF` | Wordmark "eura" (version dark) |

**Règle** : le cyan est réservé au signe N. Il ne colore jamais "eura".

## Typographie wordmark
- Police : Inter (grotesque géométrique)
- Casse : tout en bas de casse
- Poids : SemiBold (600)
- Tracking : serré (-3 units)
- L'épaisseur optique = épaisseur des jambages du N

## Interdits absolus
- Zéro point décoratif, nœud, synapse
- Zéro courbe organique main levée
- Zéro effet 3D, ombre, dégradé, glow
- Zéro élément non justifiable sur grille
